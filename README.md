## Install project

1. First, install dependencies:
```bash
npm install
# or
yarn install
```

2. Run docker compose
```bash
docker compose up -d

```

3. Rename ```.env.template``` for ```.env```


## Run project
First, run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
